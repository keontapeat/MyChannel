## 🚀 Getting Started

### Prerequisites
- **Xcode 15.0+**
- **iOS 17.0+**
- **Swift 5.9+**

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/MyChannel.git
   cd MyChannel
   ```

2. Open in Xcode:
   ```bash
   open MyChannel.xcodeproj
   ```

3. Build and run (⌘+R)
